# SimpleDuelArena
## 概要

[アジ鯖](https://azisaba.net/)のミニ鯖 Miz-PvPサーバー用に制作したDuelプラグイン。

コマンドとコンフィグファイルからアリーナの位置などを編集できる(ようにする予定)。
